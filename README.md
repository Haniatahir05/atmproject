## ATM Simulation System
A console-based Java application that simulates the core functionalities of an Automated Teller Machine (ATM). This project demonstrates the use of Object-Oriented Programming (OOP) to handle user accounts, secure authentication, and financial transactions.
## Features
* User Authentication: Secure login system using a User ID and PIN.
* Balance Inquiry: Real-time checking of current account funds.
* Cash Withdrawal: Ability to deduct funds from the account with validation for sufficient balance.
* Cash Deposit: Seamlessly add funds to the account.
* Transaction History: (Optional/Included) View a log of recent activities.
## Technical Implementation
The project is contained within a single robust file: AtmProject.java. Key programming concepts used include:
* Switch-Case Logic: For handling the main menu navigation.
* Conditional Statements: To validate PINs and check for sufficient withdrawal amounts.
* Scanner Class: For capturing user input via the console.
* Loops: To keep the session active until the user chooses to exit.

## Getting Started
### Prerequisites
* Java Development Kit (JDK) 8 or higher installed.
* An IDE (IntelliJ, Eclipse, VS Code) or terminal.
### Installation & Execution
1. Clone the repository:
```Bash```
git clone https://github.com/Haniatahir05/atmproject.git
```Bash```
2. Compile the program:
```Bash```
javac AtmProject.java
```Bash```
3. Run the application:
```Bash```
java AtmProject
```Bash```
## Usage Flow
1. Run the program.
2. Enter your User ID and PIN.
3. Choose an option from the menu:
     * 1 - Check Balance
     * 2 - Deposit Money
     * 3 - Withdraw Money
     * 4 - Exit
4. Follow the on-screen prompts to complete your transaction.
